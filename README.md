# -Oh-hi-project
A funny website of greeting in many countries. You can see how it would be on different languages and know funny or any facts!

## Usage
Open `index.html` in your web browser to view the website. Select a country from the dropdown to see the greeting in that country's language and a fun fact about the country.

Available at: https://ruslan-alimov.github.io/-Oh-hi-project/
